1. Regresion Lineal

Implementado
Predice valores continuos usando una linea recta. Usa gradiente descendente para minimizar el error cuadratico medio. Conceptos: derivadas (gradiente), limites (convergencia).
h(x) = theta_0 + theta_1 * x
J(theta) = (1/2m) * sum(h(x_i) - y_i)^2
dJ/dtheta_j = (1/m) * sum(h(x_i) - y_i) * x_i

2. Regresion Logistica

Implementado
Clasificacion binaria (ventas buenas/malas, clientes que se van/que se quedan). Usa la funcion sigmoide y su derivada. Conceptos: derivadas (sigmoide), limites (cuando x tiende a +/- infinito).
g(z) = 1 / (1 + e^-z)
h(x) = g(theta_0 + theta_1 * x)
J(theta) = -sum(y * log(h) + (1-y) * log(1-h))

3. K-Means

Implementado
Clustering: segmenta negocios o productos en grupos. Conceptos: limites (convergencia cuando las iteraciones tienden a infinito), distancia euclidiana (optimizacion).
J = sum ||x_i - mu_c(i)||^2
Asignar c(i) = argmin_j ||x_i - mu_j||^2
Actualizar mu_j = (1/|C_j|) * sum x_i
